phenochar2.0 (ver. 2)

phenochar2.0

phenochar2.0 is a software package for extracting phenotypic characters from BDML version 0.18 files.

Author

SSBD Group (ssbd at riken.jp)

History

2014/03/27: Ver. 1 released
2015/11/27: Ver. 2 released (Counting function added)

Source

phenochar2.0v2.zip

Installation

  1. Download phenochar2.0v2.zip and unzip the file.
  2. Install codesynthesis XSD.
  3. Set environmental variable XSD_ROOT to root directory of the codesynthesis XSD.
  4. make the files.

Description

These programs extract phenotypic characters from the content described in BDML version 0.18 file.

Function

  • angle calculates angle among three components.
  • centroid calculates centroid of components.
  • distance calculates distance between two components.
  • move calculates migration length.
  • timelength calculates existing time of a component.
  • volume calculates volume of components.
  • count calculates the number of components.
