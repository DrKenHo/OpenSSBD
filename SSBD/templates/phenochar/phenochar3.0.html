phenochar3.0 (ver. 1)

phenochar3.0

phenochar3.0 is a software package for extracting phenotypic characters from BDML version 0.2 files.

Author

SSBD Group (ssbd at riken.jp)

History

2014/08/28: Ver. 1 released

Source

phenochar3.0v1.zip

Installation

  1. Download phenochar3.0v1.zip and unzip the file.
  2. Install codesynthesis XSD.
  3. Set environmental variable XSD_ROOT to root directory of the codesynthesis XSD.
  4. make the files.

Description

These programs extract phenotypic characters from the content described in BDML version 0.2 file.

Function

  • angle calculates angle among three components.
  • centroid calculates centroid of components.
  • distance calculates distance between two components.
  • move calculates migration length.
  • timelength calculates existing time of a component.
  • volume calculates volume of components.
