Introduction to OpenSSBD

OpenSSBD for Systems Science of Biological Dynamics is a database engine that provides a rich set of resources for storing quantitative biological data. These quantitative numerical data are represented in a new Biological Dynamics Markup Language (BDML). The new data format allows users to exchange, store, compare and analyze data through the OpenSSBD database. Users can download quantitative biological dynamics data directly in BDML format from the SSBD database. A range of software tools and applications for visualizing and analyzing quantitative biological dynamical data are developed through a set of SSBD APIs.
