About SSBD-OMERO.insight-ij

SSBD-OMERO.insight-ij is a plugin for ImageJ software on Microsoft Windows platform. The ImageJ plugin enables reading and visualizing movies and images directly from SSBD. Movies and images are managed by OMERO.web within SSBD. This plugin requires user to input a server address and a public session ID in the on-screen login form. The public session ID is providing on the top of the login page. The server address is "http://ssbd1.qbic.riken.jp". This plugin was created by modifying the original OMERO.insight-ij, which can be download from OMERO 5.0.4 plugin downloads.
