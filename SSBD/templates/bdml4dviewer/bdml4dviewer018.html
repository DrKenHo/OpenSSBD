BDML4DViewer (v0.18)

BDML4DViewer018

BDML4DViewer018 is a pulugin of ImageJ software. The plugin enables reading and visualizing content of BDML version 0.18 files.

Author

SSBD Group (ssbd at riken.jp)

History

2014/03/20: First version

Source

BDML4DViewer018_.zip

Installation

Download BDML4DViewer018_.jar to the plugins folder and restart ImageJ. Java Binding of the OpenGL (JOGL) needs to be installed.

Description

This plugin enables visualization of content described in BDML version 0.18 file.

Function

  • View angle of objects can be changed by the mouse to drag-and-move.
  • Zoom-in and zoom-out can be accomplished by up and down arrow keys.
  • Time frame can be changed by right and left arrow keys.
  • Time frame can be also changed by using a slider.
